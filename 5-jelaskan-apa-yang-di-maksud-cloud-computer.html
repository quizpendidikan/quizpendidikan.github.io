3. apa yang dimaksud cloud computing?dan jelaskan skema dan model bisnis cloud technology​


Cloud computing ialah gabungan pemanfaatan teknologi komputer dalam suatu jaringan pengembangan berbasis internet yang memiliki fungsi untuk menjalankan program maupun aplikasi melalui komputer-komputer yang terkoneksi pada waktu yang sama, akan tetapi tidak semua yang terkoneksi dengan internet menggunakan cloud computing.

Pembahasan  

Manfaat dari teknologi cloud computing :

Semua data tersimpan di server secara terpusat

Keamanan data

Fleksibilitas dan skabilitas yang tinggi

Investasi jangka panjang

Skema & model bisnis layanan cloud computing

IaaS (Infrastructure as a Service) : model komputasi awan dimana penyedia layanan memberikan jasa organisasi infrastruktur seperti server, hardware jaringan serta data stirage yang bisa digunakan konsumen.

PaaS (Platform as a Service) : model komputasi awan yang didesain agar memberikan developer sebuah platform yang memungkinkan developer tersebut untuk membuat dan mengembangkan aplikasi baru diatasnya.

Saas (Software as a Service) : model layanan komputasi awan yang ada dalam bentuk aplikasi yang diap untuk dipakai.

5. apakah yang di maksud dengan cloud hosting?​


Memahami Pengertian dan Arsitektur Cloud Hosting adalah tipe web hosting yang menggunakan berbagai macam server untuk menyeimbangkan load dan memaksimalkan uptime. Alih-alih menggunakan single server, website Anda akan mengakses “cluster” yang memanfaatkan resource dari pool yang terpusat

Jika masih awam, pastinya Anda akan bertanya-tanya, apa itu cloud hosting? Memahami pengertian cloud hosting tidak begitu sulit. Cloud hosting adalah tipe web hosting yang menggunakan berbagai macam server untuk menyeimbangkan load dan memaksimalkan uptime. Alih-alih menggunakan single server, website Anda akan mengakses “cluster” yang memanfaatkan resource dari pool yang terpusat. Hal ini menunjukkan bahwa kalau ada satu server yang gagal, maka server yang lain tidak akan terpengaruh, melainkan tetap berfungsi.

Misalkan saja cloud sebagai web dari beberapa komputer berbeda dan semuanya saling terhubung. Semakin banyak mesin yang terhubung ke jaringan, maka semakin besar pula resource yang ditambahkan ke cloud secara keseluruhan.

Di cloud hosting, Anda akan mendapatkan suatu bagian yang disebut cloud cluster. Hal ini berbeda dari web hosting pada umumnya, di mana Anda memperoleh space dengan jumlah tertentu dari single server.

Beberapa keunggulan cloud hosting adalah fokus utama pada uptime, resource sendiri, kemudahan dalam scaling, dan alamat IP yang dedicated.

17. 4. Jelaskan yang dimaksud dengan cloud storage danJelaskan perbedaan dan persamaannya dengan aplikasi berbasiscloud​


Saat Anda menyimpan foto, musik, film, atau jenis data online lainnya daripada menggunakan penyimpanan fisik Anda sendiri (hard disk, kartu SD, dll.) Untuk melakukannya, Anda menyimpan data di cloud. Aplikasi seperti iCloud, Google Drive, Dropbox adalah semua contoh penyimpanan cloud.

- Ketika Anda menggunakan layanan online untuk melakukan tugas-tugas spesifik di mana sumber daya komputer lain digunakan - itu bisa melakukan pencarian Google, Netbanking, atau bahkan memvoting jawaban di Quora - Anda menggunakan layanan berbasis cloud untuk menyelesaikan tugas Anda .

Cloud computing adalah ketika pekerjaan komputasi dilakukan oleh komputer lain, komputer yang bisa berjarak beberapa mil dari lokasi Anda, untuk melakukan / menyelesaikan tugas tertentu. Media sosial adalah metode komunikasi berbasis cloud. Netbanking adalah metode perbankan berbasis cloud. Belanja online adalah metode belanja berbasis cloud. YouTube, Netflix, Gmail, Paypal, Spotify adalah beberapa layanan berbasis cloud yang digunakan orang setiap hari.

Komputasi dan penyimpanan cloud bukan dua konsep yang sepenuhnya berbeda. Layanan berbasis cloud memang memanfaatkan data yang disimpan, sama seperti bagaimana komputer Anda menggunakan hard drive Anda.

Bayangkan itu seperti menggunakan komputer Anda di mana RAM, prosesor dan hard drive ada di tempat lain. Anda telah menghubungkan mereka menggunakan internet dan kemudian menggunakannya. Itu hampir sama, hanya pada skala yang lebih besar.

18. 4. Jelaskan yang dimaksud dengan cloud storage dan Jelaskan perbedaan dan persamaannya dengan aplikasi berbasis cloud​


Cloud storage adalah layanan penyimpanan data yang disediakan oleh penyedia layanan cloud. Layanan ini memungkinkan pengguna untuk menyimpan, mengelola, dan mengakses data mereka secara online dari mana saja dan kapan saja dengan menggunakan koneksi internet.

Perbedaan antara cloud storage dan aplikasi berbasis cloud adalah pada fungsinya. Cloud storage hanya berfungsi sebagai tempat penyimpanan data, sedangkan aplikasi berbasis cloud memiliki fungsi tambahan seperti pengolahan data, pengeditan, dan berbagi data dengan pengguna lainnya.

Namun, keduanya memiliki persamaan dalam hal pengaksesannya yang sama-sama menggunakan koneksi internet dan dapat diakses dari mana saja dan kapan saja. Selain itu, keduanya juga memberikan kemudahan dalam berbagi data dengan orang lain.

27. 4. Jelaskan yang dimaksud dengan cloud storage danJelaskan perbedaan dan persamaannya dengan aplikasi berbasiscloud


Cloud Storage: Cloud storage merujuk pada penyimpanan data dan file di server yang terhubung ke internet, biasanya di fasilitas data center yang dikelola oleh penyedia layanan cloud. Pengguna dapat mengunggah, menyimpan, dan mengakses data mereka melalui koneksi internet, tanpa perlu penyimpanan fisik lokal. Beberapa contoh penyedia layanan cloud storage terkenal adalah Google Drive, Dropbox, dan Amazon S3.

Perbedaan dan Persamaan dengan Aplikasi Berbasis Cloud:

Perbedaan:

1. Fungsi Utama: Cloud storage digunakan untuk menyimpan dan mengelola data, seperti file dan dokumen, sedangkan aplikasi berbasis cloud adalah perangkat lunak yang dijalankan dalam lingkungan cloud untuk menjalankan tugas-tugas tertentu, seperti pengolah kata, spreadsheet, atau komunikasi.

2. Jenis Data: Cloud storage fokus pada penyimpanan data, sementara aplikasi berbasis cloud digunakan untuk memproses data, mengolahnya, dan membuat dokumen atau informasi baru.

3. Akses Data vs. Pengolahan Data: Dalam cloud storage, Anda menyimpan data dan kemudian dapat mengaksesnya dari berbagai perangkat, tetapi tidak melakukan pengolahan data secara langsung. Dalam aplikasi berbasis cloud, data diolah untuk mencapai tujuan tertentu, seperti mengedit dokumen.

Persamaan:

1. Koneksi Internet: Kedua cloud storage dan aplikasi berbasis cloud memerlukan koneksi internet untuk mengakses dan menggunakan layanan tersebut.

2. Pengelolaan Data: Kedua jenis layanan melibatkan manajemen data, meskipun dalam konteks yang berbeda. Cloud storage memungkinkan pengguna untuk mengelola dan menyimpan data mereka, sedangkan aplikasi berbasis cloud memungkinkan pengguna untuk mengelola dan memproses data mereka.

3. Fleksibilitas dan Akses dari Berbagai Perangkat: Kedua jenis layanan ini memungkinkan akses data dan aplikasi dari berbagai perangkat, seperti komputer, tablet, atau ponsel.
