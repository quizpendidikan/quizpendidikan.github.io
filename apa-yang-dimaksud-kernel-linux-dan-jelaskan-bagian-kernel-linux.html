1. apa yg di maksud kernel linux? sebutkan bagian bagian dari kernel Linux tersebut


Kernel Linux adalah kernel yang digunakan dalam sistem operasi GNU/Linux. Kernel ini merupakan turunan dari keluarga sistem operasi UNIX, dirilis dengan menggunakan lisensi GNU General Public License (GPL), dan dikembangkan oleh pemrogram di seluruh dunia. Linux merupakan contoh utama dari perangkat lunak bebas dan sumber terbuka.

Kernel Linux terdiri dari beberapa bagian penting, seperti : manajemen proses, manajemen memori, hardware devicedrivers, ?lesystem drivers, manajemen jaringan dan lain-lain. Namun bagian yang terpenting ialah manajemen prosesdan manajemen memori. Manajemen memori menangani daerah pemakaian memori, daerah swap, bagian-bagiankernel dan untuk buffer cache.

15. Apa yang dimaksud dengan EOL dan LTS dalam kernel Linux​


Jawaban:

EOL = End of Life

LTS = Long Term Service

Penjelasan:

End of Life (EOL)

Kernel Linux dapat mencapai "End of Life" setelah beberapa revisi perbaikan bug, yang berarti bahwa pengelola kernel tidak akan merilis lagi perbaikan bug untuk versi kernel ini

Long Term Service (LTS)

Kernel Linux dapat dimasukkan ke dalam pemeliharaan "jangka panjang" (LTS), yang berarti bahwa pengelola akan menyediakan perbaikan bug untuk revisi kernel ini untuk jangka waktu yang lebih lama.

Jika versi kernel yang Anda gunakan ditandai "EOL", Anda harus mempertimbangkan untuk memutakhirkan ke versi utama berikutnya karena tidak akan ada lagi perbaikan bug yang disediakan untuk versi kernel yang Anda gunakan.

b. Repository adalah sekumpulan paket-paket aplikasi atau program untuk sebuah sistem operasi linux yang digunakan untuk menunjang kinerja dari sebuah aplikasi, program, dan sebagainya yang didapatkan dari Server Mirror atau CD/DVD atau media penyimpanan lainnya.

c. Kernel adalah ketika komputer dihidupkan pertama kali (powered on), maka komputer tersebut akan memanggil sistem operasi dari hard disk melalui RAM. Bagian dari sistem operasi yang berinteraksi langsung dengan perangkat keras komputer.

d. Terminal Linux adalah seperti Command Line di windows, jadi suatu perintah yang secara langsung ditujukan ke Linux dengan cara mengetikan secara manual (tanpa GUI).

e. Mode GUI adalah antarmuka pada sistem operasi atau komputer yang menggunakan menu grafis agar mempermudah para pengguna-nya untuk berinteraksi dengan komputer atau sistem operasi.

f. Mode CLI adalah antarmuka pada sistem operasi atau komputer yang menggunakan menu baris perintah atau text atau ketikkan dari keyboard untuk berinteraksi denga sistem operasi atau komputer tersebut.

g. Virtualisasi adalah merupakan sustu metode instalasi sistem operasi yang banyak digunakan, karena mempunyai beberapa kelebihan dibandingkan dengan sistem lain. Salah satu kelebihannya adala menghemat resources terutama hardware.

h. Multibooting adalah suatu kondisi dimana dalam satu buah komputer terdapat sistem operasi lebih dari satu sistem operasi.
