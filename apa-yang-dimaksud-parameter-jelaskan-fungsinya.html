Yang dimaksud dengan parameter MYSQLI_ASSOC, MYSQLI_NUM, dan MYSQLI_BOTH adalah parameter yang digunakan dalam bentuk array. Perbedaan ketiga parameter tersebut hanya terletak pada cara array tersebut didefinisikan.


Pada MYSQLI_NUM = Item array akan menggunakan indeks key bertipe numerik.

Pada MYSQLI_ASSOC = Item array akan menggunakan nama kolom sebagai index key

Pada MYSQLI_BOTH = Item array akan diduplikasikan, yang satu menggunakan indeks key bertipe numerik, dan yang lainnya menggunakan nama kolom sebagai index key


Parameter yang digunakan dalam fungsi mysqli_fetch_rows adalah sebagai berikut

mysqli_fetch_row(result);


Result wajib diisi. Menandakan kumpulan hasil identifier yang dikembalikan oleh mysqli_query(), mysqli_store_result() atau mysqli_use_result().


mysqli_fetch_rows digunakan untuk mengambil sebuah baris dari kumpulan hasil dan mengembalikannya ke dalam bentuk array yang telah dicacah

19. Jelaskan parameter parameter tingkat pelaksanan demokrasi dalam suatu negara


PARAMETER DEMOKRASI
Prinsip-prinsip Negara demokrasi yang telah tersebutkan di atas kemudian dituangkan dalam konsep yang praktis untuk dapat diukur dan dicirikan. Ciri-ciri inilah yang kemudian dijadikan parameter untuk mengukur tingkat pelaksanaan demokrasi suatu Negara dalam menjalankan tata pemerintahanya sehingga dikatakan demokratis atau tidak, ada 4 aspek dalam mengukur hal ini, yaitu:
1) masalah pembentukan Negara.
2) dasar kekuasaan Negara.
3) susunan kekuasaan Negara.
4) masalah kontrol rakyat.
Menurut Djuanda Wijaya parameter kehidupan demokratis adalah sebagai berikut:
a. dinikmati dan dilaksanakan hak serta kewajiban politik oleh masyarakat berdasarkan prinsip-prinsip dasar HAM yang menjamin adanya kebebasan, kemerdekaan, dan rasa merdeka.
b. penegakan hukum yang mewujud pada supremasi hukum.
c. kesamaan hak dan kewajiban anggota masyarakat.
d. kebebasan pers yang bertanggung jawab.
e. pengakuan pada hak minoritas.
f. pembuatan kebijakan Negara yang berlandaskan asas pelayanan, pemberdayaan, dan pencerdasan.
g. system kerja yang kooperatif dan kolaboratif.
h. keseimbangan dan keharmonisan. 
i. tentara yang professional.
j. lembaga peradilan yang independent.
Amien Rais menambahkan kriteria lain dalam parameter demokrasi, yaitu:
a) adanya partisipasi dalam pengambilan keputusan. 
b) distribusi pendapatan secara adil.
c) kesempatan memperoleh pendidikan.
d) ketersediaan dan keterbukaan informasi.
e) mengindahkan fatsoen politik.
f) kebebasan individu.
g) semangat kerjasama.
h) hak untuk protes.
Pendapat berikutnya adalah pendapat dari Sri Soemantri yang menyatakan bahwa:
1. hukum diterapkan dengan persetujuan wakil rakyat yang dipilih secara bebas.
2. hasil pemilu dapat menyebabkan pergantian orang-orang dalam pemerintahan.
3. pemerintahan harus terbuka.
4. kepentingan minoritas harus dipertimbangkan.
Frans Magnis Suseno juga berpendapat bahwa parameter Negara demokrasi adalah:
1. Negara terikat hukum.
2. kontrol efektif pemerintahan oleh rakyat.
3. pemilu yang bebas.
4. adanya jaminan terhadap hak-hak minoritas.
Sedangkan menurut G. Bingham Powell Jr. menurutnya parameter Negara demokratis adalah:
1. pemerintah mengklaim mewakili hasrat para warganya.
2. klaim berdasarkan pada adanya pemilihan kompetitif secara berkala antara calon alternative.
3. partisipasi warga dewasa sebagai calon dipilih dan pemilih.
4. pemilihan bebas.
5. adanya kebebasan dasar warga negaranya.
Selanjutnya Affan Ghofar (pakar politik UGM) sejumlah prasyarat untuk mengamati apakah sebuah political order (pemerintahan) merupakan system yang demokratik atau tidak melalui ukuran:
1. akuntabilitas.
2. rotasi kekuasaan. 
3. recruitment politik.
4. pemilihan umum. 
5. adanya pengakuan dan perlidungan hak-hak dasar.
Kelima elemen tersebut berlaku secara universal di dalam melihat demokratis tidaknya suatu rezim pemerintahan (political order).

   cout << "Hello, " << nama << endl;

}

void = sebuah tipe dari fungsi, lebih tepatnya fungsi tanpa kembalian.

sapa = sebagai nama fungsi yang telah kita buat dan akan digunakan untuk memanggil fungsi itu nantinya di main program.

{} = menandakan kode yang ada di dalam tanda kurung kurawal ini sebagi bagian dari milik fungsi sapa.

nama = sebagai paramater.

cout = berarti console out, atau sederhannya menampilkan hasil dari program ke console/terminal.

<< = tanda yang berati akan mengeluarkan/menampilkan ke terminal

"Hello" = teks/string biasa.

nama = akan menampilkan nama(Parameter) ke console/terminal.

endl = bisa diartikan sebagia "\n" atau disebut juga dengan new line/baris baru.

; = titik dua/semicolom berarti sebagai penanda jika kode pada baris ini akan berhenti.

#include <iostream>

using namespace std;

using namespace std; =  membuat kita tidak perlu menggunakan std:: lagi di dalam program.

int main() = sebagi main program.

{} = menandakan kode yang ada di dalam tanda kurung kurawal ini sebagi bagian dari milik main program.

str nama; = sebagai variabel yang menyimpan data/value yang dimasukan oleh user.

cout << =  berarti console out, atua sederhannya menampilkan hasil dari programnya ke console/terminal.

; = berarti sebagai penanda jika kode pada baris ini akan berhenti.

cin >> = console input, yang berarti user akan memasukan/input data ke variabel nama;.

sapa(nama); = memnaggil fungsi. kita akan mengoper data yang telah di-input-kan oleh user di variabel nama; ke dalam fungsi untuk natinya ditampilkan ke dalam console/terminal.

; = berarti sebagai penanda jika kode pada baris ini akan berhenti.

8. Function Parameter dan Argument. Function Parameter adalah sebuah variabel yang berada di dalam sebuah fungsi dan  menjadi tempat penyimpanan nilai dari argument yang diberikan saat pemanggilan function. sedangkan Argument adalah parameter yang dimasukan ketika melakukan pemanggilan fungsi. Ini merupakan tempat dimana kita bisa memberikan data untuk digunakan di dalam fungsi yang dipanggil dan kemudian akan diolah lebih lanjut di dalam fungsi.
