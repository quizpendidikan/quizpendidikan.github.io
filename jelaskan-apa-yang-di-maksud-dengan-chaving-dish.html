33. data yang tersimpan dalam sebuah flash dish berbentuk

materi : hardware
kata kunci : perangkat keras, data, flashdisk
kode soal : 11
kode kategorisasu : 10.11

pembahasan :
data yang tersimpan dalam sebuah flashdisk dapat berbentuk gambar, video, audio, file berekstensi tidak dikenal, bahkan shortcut dan virus. sebenarnya bisa juga untuk bootable sistem operasi. tetapi ini akan menghapus semua datamu.

jika flashdiskmu tidak dapat menyimpan file yang disebutkan diatas bisa jadi flashdiskmu rusak/korup. jika hal ini terjadi ada cara-cara untuk mengatasinya:
1. coba format flashdisknya (backup datanya dulu)
2. lakukan pengecekan kesehatan flashdisk
3. jika tidak bisa, waktunya untuk mengganti flashdisknya (lembiru)
sebenarnya, flashdisk juga ada ciri yang tidak biasa dan ciri ini merupakan ciri flashdisk palsu
