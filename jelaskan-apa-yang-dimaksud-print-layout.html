1. Apa yang dimaksud print layout


perangkat dan teknologi kamusPrint layout adalah salah satu mode dalam Microsoft Word yg berguna untuk menampilkan dokumen pada Microsoft WOrd sebagaimana akan tampak dokumen tersebut ketika dicetak.

2. jelaskan perbedaan antara print layout, web layout dan fullscreen


- Print Layout : Artinya adalah tampilan sesuai dengan kertas dari Printer yang akan digunakan.

- Web Layout : Artinya tampilan layar akan menyesuaikan dengan design / mode tampilan sebuah web.

- Full Screen : Tampilan akan memenuhi layar
print layout = tampilan ketika dicetak
web layout = tampilan ketika dibuka di browser
fullscreen = tampilan layar penuh

Untuk mencetak hasil laporan yang di update pada Microsoft Word kamu dapat menggunakan fasilitas "print" secara langsung, baik melalui menu bar File maupun dengan jalan pintas dengan menekan tombol CTRL+P.

Pembahasan:

AKtivitas mencetak bisa kamu lakukan dengan dua cara yang berbeda, dengan menekan tombol maupun melalui jalan pintas keyboard seperti yang relatif lebih cepat jka dibandingkan dengan menekan tombol pada menu bar.

Walaupun pada dasarnya kamu bisalangsung melakukan print/mencetak ada baiiknya selalu memastikan konfigurasi halaman laporan atau dokumen yang akan dicetak sudah sesuai dengan format dan media cetak yang digunakan (print preview). Misalnya seperti margin, layout dan ukuran kertasnya.

Pilih salah satu diantara klik ikon cetak atau print pada bagian kanan menu bar paling atas atau menekan tombol keyboard CTRL+PSetelah jendela  print keluar pastikan setingan/konfigurasi halaman susah sesuai dengan tampilan yang kamu ininkanJika sudah yakin klik tombol print.

30. apa fungsi dari print layout?


Kelas: X

Mata pelajaran: Teknologi Informasi

Materi: Microsoft Word

 

Saya akan mencoba menjawab dengan dua jawaban:

 

Jawaban pendek:


Print layout berguna menampilkan dokumen pada Microsoft WOrd sebagaimana akan tampak dokumen tersebut ketika dicetak.  

Jawaban panjang:  

Dalam Microsoft Word ada lima mode untuk melihat dokumen yaitu:  

Read Mode  

Dokumen mengisi jendela Microsoft Word dan diformat dalam bentuk seperti tampilan di tablet atau majalah. Tampilan ini sangat berguna untuk memeriksa dokumen, serta membaca yang telah Anda download atau terima di email.  

Print Layout  

Tampilan umum Microsoft Word. Dokumen umum, seperti memo, surat, dan laporan, sering ditulis dan diedit dalam tampilan Print Layout. Salah satu keuntungan bekerja dalam pandangan ini adalah kepatuhannya terhadap WYSIWYG (apa yang Anda lihat adalah apa yang Anda dapatkan). Margin, header / footer, dan format sesuai dengan hasil cetak. Halaman ditampilkan sebagai potongan kertas setara dengan jeda fisik antar halaman.  

Web Layout  

Tampilan Web Layout digunakan untuk membuat, melihat, dan mengedit halaman karena akan muncul secara online saat dibuka di browser. Dengan memilih Save As di mode ini, kitadapat menyimpan halaman dalam beberapa format yang kompatibel dengan web.  

Outline  

Outline akan menampilkan susunan dokumen seperti susunan bab, sub bab dan point penting dokumen.  

 Draft

 

Draft digunakan saat kecepatan sangat penting, misalnya saat menggunakan komputer yang lambat. Tampilan Draft menampilkan dokumen sebagai teks kontinu. Jeda halaman dilambangkan dengan garis putus-putus.
