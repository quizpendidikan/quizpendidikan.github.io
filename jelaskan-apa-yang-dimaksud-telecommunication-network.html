20. Jelaskan apa yang dimaksud dengan Network id dan host id dan berikan contohnya


Pada dasarnya Network ID sendiri dapat kita artikan sebagai suatu Identitas dari suatu kelompok IP Address atau subnet, yang mana Network ID ialah suatu bagian dari IP address yang menunjukan letak network atau jaringan dari suatu IP Address. Sedangkan untuk Host ID sendiri ialah suatu nomor perangkat keras fisik Ethernet Network Interface Card atau yang sering disingkat dengan sebutan NIC.

Sebagai salah satu contohnya ialah:

Kelas A yaitu 10.11.12.1 maka untuk contoh ini network ID nya ialah 10. sedangkan untuk Host ID ialah 11.12.1.Kelas B yaitu 188.10.11.1 maka untuk contoh ini network ID nya ialah 188.10. sedangkan untuk Host ID ialah 11.1.Kelas C yaitu 192.168.1.1 maka untuk contoh ini network ID nya ialah 192.168.1 sedangkan untuk Host ID ialah 1.

Pembahasan

Pada umumnya Net ID sendiri dapat berperan penting dalam mengidentifikasi suatu network dari network yang lain, sedangkan host ID berperan untuk identifikasi host dalam suatu network, yang mana pada konteks ini dapat kita simpul kan bahwa seluruh host yang tersambung dalam jaringan yang sama memiliki net ID yang sama.

34. apa yang di maksud dengan jaringan network​


Jawaban:

Network adalah jaringan dari system komunikasi data yang melibatkan sebuah atau lebih system komputer yang dihubungkan dengan jalur transmisi alat komunikasi membentuk satu system. Dengan network, komputer yang satu dapat menggunakan data di komputer yang lain.

Penjelasan:

Network atau jaringan adalah sebuah kumpulan komputer yang dihubungkan dengan kabel sehingga komputer satu dengan komputer lainnya dapat saling komunikasi, bertukar informasi sharing file dan lain-lain. Adapun jenis jaringan komputer sebagai berikut:

1. LAN (Local Area Networking) yaitu jaringan yang dibatasi oleh area yang reatif kecil dan biasanya tidak jauh dari sekitar 1 km persegi contohnya disekolah, gedung perkantoran dll.

2. MAN (Metropolitan Area Networking) yaitu jaringan yang biasanya meliputi area yang lebih besar dari LAN seperti Propinsi

3. WAN (Wide Area Networking) yaitu jaringan yang yang lingkupnya biasanya sudah menggunakan sarana Satelit ataupun kabel bawah laut sebagai contoh keseluruhan jaringan BANK BNI yang ada di Indonesia ataupun yang ada di Negara-negara lain.

Jaringan network atau jaringan komputer adalah jaringan telekomunikasi yang memungkinkan antar komputer untuk saling berkomunikasi dengan cara bertukar data.
