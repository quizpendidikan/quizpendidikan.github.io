20. misalkan akar-akar persamaan kuadrat 3x^2 + 3x - 5 = 0 adalah alfa dan beta Tentukan persamaan kuadrat yang akar-akarnya 2A + 3 dan 2B + 3 =tolong bantu jawab yah soalnya mau dikumpulkan,dijawab beserta langkah langkahnyaterimakasih​


Jawab:

Penjelasan dengan langkah-langkah:

Untuk menentukan persamaan kuadrat yang akar-akarnya 2A + 3 dan 2B + 3, kita harus mengingat bahwa persamaan kuadrat dalam bentuk umum adalah ax^2 + bx + c = 0, dan akar-akar persamaan tersebut dapat ditentukan dengan menggunakan rumus akar-akar persamaan kuadrat yaitu x = (-b ± √(b^2 - 4ac))/2a.

Dalam kasus ini, persamaan kuadrat yang diberikan adalah 3x^2 + 3x - 5 = 0. Sehingga a = 3, b = 3, dan c = -5.

Akar-akar persamaan ini ditentukan dengan menggunakan rumus akar-akar persamaan kuadrat:

x = (-3 ± √(3^2 - 4 * 3 * -5))/2 * 3

x = (-3 ± √(9 + 60))/6

x = (-3 ± √69)/6

Dari sini, kita dapat mengetahui bahwa akar-akarnya adalah x = ( √69 + 3 )/6 = alfa dan x = (- √69 + 3 )/6 = beta

Untuk menentukan persamaan kuadrat yang akar-akarnya 2A + 3 dan 2B + 3, kita harus mengetahui bahwa dalam persamaan kuadrat, akar-akar persamaan kuadrat merupakan kali dari x, maka x = alpha = ( √69 + 3 )/6 dan x = beta = (- √69 + 3 )/6.

22. Jika a dan b adalah akar-akar dari persamaan 2x^2-x-3=0 maka tentukan persamaan kuadrat baru yang akar-akarnya a+2 dan b+2 Bantu jawab yah kak soalnya harus dikumpul besok


Operasi Akar-Akar Persamaan Kuadrat

Misal, m dan n adalah akar-akar dari persamaan kuadrat px² + qx + r = 0, maka

m + n = [tex]-\frac{q}{p}\\[/tex]

m . n = [tex]\frac{r}{p}\\[/tex]

[tex]\\[/tex]

Menentukan persamaan kuadrat baru:

Misal, m+k dan n+k adalah akar-akar persamaan kuadrat

maka, persamaan kuadrat barunya adalah

x² - [(m+k)+(n+k)] x + (m+k)(n+k) = 0

[tex]\\[/tex]

Diketahui:

PK: 2x² - x - 3 = 0

a dan b adalah akar-akar persamaan kuadrat 2x² - x - 3 = 0, maka

a + b = [tex]-\frac{-1}{2}\\[/tex] = [tex]\frac{1}{2}\\[/tex]

a . b = [tex]\frac{-3}{2}\\[/tex] = [tex]-\frac{3}{2}\\[/tex]

[tex]\\[/tex]

a + 2 dan b + 2 adalah akar-akar persamaan kuadrat baru, maka

(a + 2) + (b + 2)

= (a + b) + 4

= [tex]\frac{1}{2}\\[/tex] + 4

= [tex]\frac{1}{2}\\[/tex] + [tex]\frac{8}{2}\\[/tex]

= [tex]\frac{9}{2}\\[/tex]

[tex]\\[/tex]

(a + 2)(b + 2)

= ab + 2a + 2b + 4

= ab + 2(a + b) + 4

= [tex]-\frac{3}{2}\\[/tex] + 2([tex]\frac{1}{2}\\[/tex]) + 4

= [tex]-\frac{3}{2}\\[/tex] + 1 + 4

= [tex]-\frac{3}{2}\\[/tex] + 5

= [tex]-\frac{3}{2}\\[/tex] + [tex]\frac{10}{2}\\[/tex]

= [tex]\frac{7}{2}\\[/tex]

[tex]\\[/tex]

Persamaan kuadrat baru:

x² - [(a+2)+(b+2)] x + (a+2)(b+2) = 0

x² - [tex]\frac{9}{2}\\[/tex]x + [tex]\frac{7}{2}\\[/tex] = 0

2x² - 9x + 7 = 0

[tex]\\[/tex]

Jawab:

﻿x² + 11x + 17 = 0

Penjelasan dengan langkah-langkah:

﻿ax² + bx + c = 0

﻿x² + 5x - 7 = 0

x﻿₁ + x₂ = -b/a = -5/1 = -5

﻿x₁x₂ = c/a = -7/1 = -7

﻿α = x₁ - 3

﻿β = x₂ - 3

﻿x² - (α + β)x + αβ = 0

﻿x² - (x₁ - 3 + x₂ - 3)x + (x₁ - 3)(x₂ - 3) = 0

﻿x² - (x₁ + x₂ - 6)x + x₁x₂ - 3(x₁ + x₂) + 9 = 0

﻿x² - (-5 - 6)x - 7 - 3(-5) + 9 = 0

﻿x² + 11x + 17 = 0

﻿Cara cepat

﻿Persamaan kuadrat ax² + bx + c = 0 akar-akarnya x₁ atau x₂. Persamaan kuadrat yang akar-akarnya x₁ - m atau x₂ - m adalah a(x + m)² + b(x + m) + c = 0

﻿1(x + 3)² + 5(x + 3) - 7 = 0

﻿x² + 6x + 9 + 5x + 15 - 7 = 0

﻿x² + 11x + 17 = 0
