Persamaan linear

Persamaan linear merupakan suatu persamaan aljabar, dimana setiap sukunya mengandung konstanta, atau juga perkalian konstanta dengan suatu variable tunggal. Persamaan linear dikatakan linear sebab ketika persamaan ini digambarkan dalam bentuk grafik koordinat kartesius akan menghasilkan suatu garis lurus.

Pengertian sistem persamaan linear

sistem persamaan linear merupakan kumpulan persamaan linear yang dibangun oleh beberapa variable. Sistem persamaan linear terdiri dari jumlah variabelnya. jika terdapat dua variable misalkan x dan y maka sistem persamaan tersebut dinamakan sistem persamaan linear dua variable dan seterusnya untuk tiga atau lebih variable.  

Bentuk umum dari persamaan linear dapat dinyatakan sebagai berikut :

y=mx+b

Contoh dari bentuk persamaan linear yaitu :

Y= x + 2 (sistem persamaan linear satu variabel)

Y=2p +3q -1 (sistem persamaan linear dua variabel)

Demikian lah penjabar dari kakak, semoga dengan ilmu yang sedikiti ini dapat bermanfaat untuk adik adik semua.
