7. Apakah fungsi yang dikenai operasi limit merupakan fungsi aljabar


Jawaban:

Ya, fungsi yang dikenai operasi limit dapat menjadi fungsi aljabar. Fungsi aljabar adalah fungsi yang dapat dinyatakan dalam bentuk polinomial atau rasional. Polinomial terdiri dari suku-suku dengan koefisien bilangan riil dan variabel yang diangkat pada eksponen bilangan bulat non-negatif. Contoh fungsi aljabar adalah f(x) = 2x^3 - 5x^2 + 3x + 1.

Operasi limit sendiri adalah alat matematika yang digunakan untuk mempelajari perilaku fungsi saat variabel mendekati suatu nilai tertentu. Limit dapat diterapkan pada berbagai jenis fungsi, termasuk fungsi aljabar. Dalam konteks ini, fungsi yang dikenai operasi limit dapat berupa fungsi aljabar maupun jenis fungsi lainnya seperti fungsi trigonometri, fungsi eksponensial, dan sebagainya

26. bagaimana cara mengejakan soal operasi hitung pada aljabar ?


pada penjumlahan
*komutatif= a+b=b+a
*asosiatif=a+(b+c)=(a+b)+c
*distributif=ab+ac=a(b+c)=(b+c)a

operasi penjumlahan bentuk aljabar dapat dilaksanakan jika dan hanya jika SUKU SUKUNYA SEJENIS
Agar bisa menjumlahkan dan mengurangkan dalam bentuk aljabar, variabel dan pangkatnya itu harus sama

#Penjumlahan aljabar :
1. 2a + a = 2a + 1a = 3a
2. -b + 3b = 2b
3. a + c ≠ (tidak bisa dijumlahkan karena variabel tidak sama)
4. p² + 2p² = 3p²
5. p² + p³ ≠ (tidak bisa dijumlahkan karena pangkat tidak sama)
6. x² + 2x - 3 dan -5x² - 4x + 2
[kata "dan" diubah menjadi +]
= x² + 2x - 3 + -5x² - 4x + 2
[urutkan dengan yang variabel dan pangkatnya sama]
= x² - 5x² + 2x - 4x - 3 + 2 
= -4x² - 2x + 1
7. 2p - q - 5 dan p + 7q - 10
= 2p - q - 5 + p + 7q - 10
= 2p + p - q + 7q - 5 + 10
= 3p + 6q + 5

#Pengurangan aljabar :
1. 5a - (-3a) = 5a + 3a = 8a
2. 3b - 7b = -4b
3. 2c - a ≠ (tidak bisa dikurang karena jumlah tidak sama)
4. -3p² + p² = -2p²
5. p³ - p ≠ (tidak bisa dikurang karena pangkat tidak sama
6. x² + 2x - 3 dan -5x² - 4x + 2
[kata "dan" diubah menjadi -]
= x² + 2x - 3 - (-5x² - 4x + 2)
= x² + 2x - 3 + 5x² + 4x - 2
= x² + 5x² + 2x + 4x - 3 - 2
= 6x² + 6x - 5
7. 2xy + 4x - 3y dan -3xy + x - 2y
= 2xy + 4x - 3y - (-3xy + x - 2y)
= 2xy + 4x - 3y + 3xy - x + 2y   
= 2xy + 3xy + 4x - x - 3y + 2y
= 5xy + 3x - y
8. -3xy + x - 2y dari 2xy + 4x - 3y
[kata "dari" diubah menjadi - dan dibalekkan]
= 2xy + 4x - 3y - (-3xy + x - 2y)
= 2xy + 4x - 3y + 3xy - x + 2y
[urutkan dengan yang variabel dan pangkatnya sama]
=  2xy + 3xy + 4x - x - 3y + 2x
9. y² - 5x + 4 dari 2y² - 3y + 7
= 2y² - 3y + 7 - (y² - 5y + 4)
= 2y² - 3y + 7 - y² + 5y - 4
= 2y² - y² - 3y + 5y + 7 - 4
= y² + 2y + 3

37. MatematikaMateri : Operasi Aljabar Pada Fungsi​


Jawaban No 1

A. f(x) + g(x)

= (x - 5) + (x² - 8x + 15)

= x² - 8x + x - 5 + 15

= x² - 7x + 10

B. f(x) - g(x)

= (x - 5) - (x² - 8x + 15)

= - x² + 8x + x - 5 - 15

= - x² + 9x - 20

C. f(x) . g(x)

= (x - 5) (x² - 8x + 15)

= x³ - 8x² + 15x - 5x² + 40x - 75

= x³ - 13x² + 55x - 75

D. f(x) / g(x)

= (x - 5) / (x² - 8x + 15)

= (x - 5) / (x - 5) (x - 3)

= 1 / (x - 3)

Jawaban No 2

A. f(x) + g(x)

= (2x + 3) + (2x² - 5x - 12)

= 2x² - 5x + 2x + 3 - 13

= 2x² - 3x - 10

B. f(x) - g(x)
