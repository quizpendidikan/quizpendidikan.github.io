10. jelaskan apa itu zero &first conditionaldan bagaimana cara mengerjakan soal zero&first conditional​


Jawaban:

Kalimat zero conditional

Digunakan untuk fakta ilmiah dan kebenaran/peraturan umum. Struktur: “If + present tense” untuk klausa kondisional, dan present tense untuk klausa utama.

Kalimat first conditional

Digunakan untuk membahas hal-hal yang mungkin terjadi di masa depan. Digunakan untuk menggambarkan hal-hal yang mudah terjadi atau diwujudkan. Struktur: “if + present tense” untuk klausa kondisional, dan future tense “will + infinitive” untuk klausa utama.

17. soal dan jaaaawaban conditional centences


I.) 0 conditional
1.) If you ... (press) this key, the game ...(start). 
Jawaban: If you press this key, the game starts.

2.) If you ... (use) hot iron on nylon, it ...(melt).
Jawaban: If you use hot iron on nylon, it melts.

3.) The video ... (pause) if you ... (touch) this button.
Jawaban: The video pauses if you touch this button.

4.) If water ... (be) frozen, it ... (expand).
Jawaban: If water is frozen, it expands.
II.) 1st conditional
1.) If you ... (come) home late,  ... (not, wake) us up!
Jawaban: If you come home late, don't wake us up!

2.) I ... (Call) the police if you ... (not, leave) now!
Jawaban: I will call (I'll call) the police if you don't leave now!

3.) You ... (enjoy) Hamilton: An American Musical and Les Miserable if you ... (Like) good musicals.
Jawaban: You will enjoy Hamilton: An American Musical and Les Miserable if you like good musicals.

4.) Jovan ... (come) if you ... (promise) not to talk about mathematics all the time.
Jawaban: Jovan will come if you promise not to talk about mathematics all the time.

5.) If she ... (read) the book carefully, she ... (understand) it.
Jawaban: If she reads the book carefully, she will understand it.

6.) If I ... (pass) this calculus test, I ... (not, have) to study at the weekend.
Jawaban:  If I pass this calculus test, I will not have (I won't have) to study at the weekend.

III.) 2nd Conditional
1.) She ... (take) up jogging if she ... (not, have) breathing problem.
Jawaban: She would take up jogging if she didn't have breathing problem.

2.) If I ... (be) you, I ... (not, buy) that ticket to the concert.
Jawaban: If I were you, I Would not (wouldn't) buy that ticket to the concert

3.) I ... (help) you if I ... (be) free on Sunday evening.
Jawaban: I Would (I'd) help you if I were free on Sunday evening.

4.) If I ... (be) a rich man, I ...(build) a big tall house with rooms by the dozen. (Inspirasi dari Fiddler on the roof).
Jawaban: If I were a rich man, I would (I'd) build a big tall house with rooms by the dozen.

5.) If I ... (know) the composer of Master of The House in Les Miserable, I ... (tell) you.
Jawaban: If I knew the composer of  Master of The House in Les Miserable, I would tell you.

6.) Where ... (you, choose) if you ... (can) live anywhere in the world?
Jawaban: Where would you choose if you could live anywhere in the world?

7.) ... (it) be all right if I ... (use) your calculator?
Jawaban: Would it be all right if I used your calculator?

Latihan (tanpa jawaban):
1.) If you ... (be) scared of insect, ... (not, go) to the insect room in this museum.
2.) She ... (be) furious if she (find) out the truth!
3.) Alexander Hamilton ... (live) another 30 years with Elizabeth Schuyler if Aaron Burr ... (not, shot) him between his ribs.
4.) Cosette ... (be) dead if Jean Valjean ... (not, save) her from the Thernadier's family.
5.) If I ... (be) Alexander Hamilton, I ... (write) another 25 essays about the Federalist Paper.
6.) If you ... (can) help me, I ... (be) very grateful.
7.) I ...(go) if you (not, stop) me at the airport.
8.) If I ... (switch) lives with Gavroche, I ... (die) as a martyr at the meadow of France.
9.) We ...(win) a million dollar if we finish at first place in The Amazing Race by Phil Keoghan.
10.) I ... (be) petrified if I ... (see) James from TheOdds1Out and Jaiden from JaidenAnimation on Vidcon this year.
11.)I ... (not, can) lie, I ... (miss) this place. (Inspirasi dari The Man On The Internet)

26. 50 soal serta jawaban tentang conditional sentence ?


Conditional sentence is a sentence that expresses a condition. A condition is something that can only happen IF something else occurs. Conditional sentences are made up of a dependent clause and an independent clause joined to express said condition.

Pembahasan

Conditional Sentence Type 1: It is possible and also very likely that the condition will be fulfilled.

Conditional Sentence Type 2: It is possible but very unlikely, that the condition will be fulfilled.

Conditional Sentence Type 3: It is impossible that the condition will be fulfilled because it refers to the past.

Contoh soal tentang conditional sentences yaitu

Conditional sentence type 1:

1. If we ... (not/work) harder, we ... (not pass) the exam.

2. If she ... (not go) to the meeting, I ....(not go) either.

3. If the teacher ... (give) us lots of homework this weekend, I ... (not be) happy.

4. If the children ... (not eat) soon, they ... (be) grumpy.

5. If I ... (not go) to bed soon, I ... (be) tired in the morning.

6. If John .... (drink) too much coffee, he ... (get) ill.

7. If she ...(go) to the library, she ... (study) more.

8. If they ... (not/arrive) soon, we ... (be) late.

9. If you ... (buy) the present, I ... (wrap) it up.

10. If Lucy ... (not quit) her job soon, she ... (go) crazy.

Conditional sentence type 2:

1. If she ... (have) her laptop with her, she ... (email) me.

2. If Lucy ... (have) enough time, she ... (travel) more.

3. If I ... (want) a new car, I ... (buy) one.

4. If Jose ... (not speak) good French, he ...(not move) to Paris.

5. If the children ... (be) in bed, I ... (be able to) have a bath.

6. If you ... (not be) so stubborn, we ... (not have) so many arguments!

7. If you ... (arrive) early, it ... (be) less stressful.

8. If Julie ... (like) chocolate, I ... (give) her some.

9. If Luke ... (live) in the UK, I ... (see) him more often.

10. If we ... (not be) so tired, we ... (go) out.

Conditional sentence type 3:

1. If the students ... (not be) late for the exam, they ... (pass).

2. If the weather ... (not be) so cold, we ... (go) to the beach.

3. If the baby ... (sleep) better last night, I ...(not be) so tired.

4. If we ... (tidy) our flat, we .. (not lose) our keys.

5. If Luke ... (not send) flowers to his mother, she ... (not be) happy.

6. If Julie ... (not go) to Sweden, she ... (go) to Germany.

7. If we ... (not have) an argument, we ... (not be) late.

8. If I ... (not go) to the party, I ... (not meet) Amanda.

9. If the children ... (not eat) all that chocolate, they ... (feel) sick.

10. If she ... (study) Mandarin, she ... (go) to Beijing.
