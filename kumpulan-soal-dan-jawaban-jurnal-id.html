Contoh kasus id, ego, dan superego di dunia maya:

1. Id: contohnya yaitu menonton konten negatif dan membagikan tautannya di sosial media dengan tidak peduli apakah konten yang dibagikan itu merugikan banyak orang.

2. Ego: contohnya yaitu tidak membalas komentar pedas orang lain di sosial media karena ia tahu bahwa membalas komentar tersebut akan semakin merugikan diri sendiri.

3. Superego: contohnya yaitu rasa ingin mencuri akun media sosial milik teman karena jumlah pengikut yang lebih banyak. Id, ego dan superego berpengaruh satu sama lain dalam menciptakan pola pikir manusia. Id memberi tuntutan terhadap kebutuhan alamiah manusia seperti makan, minum, dan seksual. Ego membatasinya dengan realita yang akan dihadapi oleh orang tersebut. Sedangkan superego, memperkuat keyakinan ego dengan didasari oelh nilai-nilai moral.

9. sebutkan Network ID. Brandcast ID, subnet mask danHost ID Pada: a. 1P 192.168.100.80/25b. IP 192.168.1.0/27Tolong dijawab kak​


Jawaban & Penjelasan:

Sebutkan Network ID, Broadcast ID, Subnet Mask dan Host ID pada IP 192.168.100.80/25

Untuk menghitung Host ID. /25 = 32 -25 = 7

2^7 = 128

Berarti Jumlah Host ID = 128 Host

128 host ini nantinya dikurangi 2, yaitu untuk Network ID dan Broadcast

Untuk Menghitung Network ID (Host Pertama)

Network ID pada 192.168.100.80 = 192.168.100.0

Untuk Menghitung Broadcast (Host Terakhir)

Broadcast pada 192.168.100.80/25 = 192.168.100.127

IP Address yang bisa digunakan

192.168.100.1 – 192.168.100.126

Untuk Menghitung Subnet Mask

Subnet Mask pada 192.168.100.80/25 = 256 – 128 = 128

Maka Subnetmasknya adalah = 255.255.255.128

Pengertian✍️

Network id

Network ID adalah bagian dari IP address yang menunjukkan di jaringan mana komputer tersebut berada.

Host id

Host adalah identitas dari host (interface router, workstation, dan device) yang terhubung ke jaringan.

Broadcast id

broadcast adalah alamat jaringan yang digunakan untuk mengirim ke semua perangkat yang terhubung ke jaringan komunikasi akses ganda.
