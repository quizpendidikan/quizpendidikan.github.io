Untuk menentukan nilai u21 dari barisan dengan rumus f(un) = an² + bn + c, kita perlu menggunakan informasi yang telah diberikan, yaitu nilai-nilai f(un) pada indeks 5, 10, dan 17. Dengan nilai-nilai ini, kita dapat membentuk sistem persamaan dan mencari nilai un pada indeks 21.

1. Pertama, kita akan menghitung f(u5), f(u10), dan f(u17) sesuai rumus f(un):

  - f(u5) = a(5)² + b(5) + c = 25a + 5b + c

  - f(u10) = a(10)² + b(10) + c = 100a + 10b + c

  - f(u17) = a(17)² + b(17) + c = 289a + 17b + c

2. Selanjutnya, kita memiliki tiga persamaan berdasarkan nilai-nilai f(un) di atas:

  - f(u5) = 5

  - f(u10) = 10

  - f(u17) = 17

3. Dengan menggunakan sistem persamaan ini, kita dapat mencari nilai a, b, dan c. Setelah kita menentukan nilai-nilai a, b, dan c, kita dapat menghitung f(u21) sesuai rumus f(un) untuk mendapatkan nilai u21.

Sistem persamaan yang harus dipecahkan adalah sebagai berikut:

  - 25a + 5b + c = 5

  - 100a + 10b + c = 10

  - 289a + 17b + c = 17

Setelah memecahkan sistem persamaan ini dan menemukan nilai a, b, dan c, kita dapat menghitung f(u21) = a(21)² + b(21) + c untuk mendapatkan nilai u21.

1, 3, 5, 7, 9, ...

Barisan aritmatika yang perbedaan tiap sukunya adalah 2

Barisan aritmatika tingkat dua adalah barisan bilangan dimana perbedaan setiap suku yg berdekatan membentuk suatu barisan aritmatika. Sehingga barisan selisihnya mempunyai beda yang sama.

13. Tentukan koefisien deret fourier a-2​


Jawab:

koefisien deret Fourier a-2 untuk sinyal X(t) = 2 sin wot adalah 0.

Penjelasan dengan langkah-langkah:

Untuk menentukan koefisien deret Fourier (a-n) dari sinyal periode T dengan persamaan X(t), kita dapat menggunakan rumus berikut:

a-n = (1/T) * integral dari (T/2) ke (-T/2) [X(t) * exp(-j * n * w0 * t)] dt

Di sini, w0 adalah frekuensi dasar sinyal dan j adalah bilangan imajiner.

Dengan menggunakan rumus di atas, kita dapat menentukan koefisien deret Fourier a-2 dari sinyal X(t) = 2 sin wot sebagai berikut:

a-2 = (1/T) * integral dari (T/2) ke (-T/2) [2 sin wot * cos (-2 * w0 * t)] dt + (1/T) * integral dari (T/2) ke (-T/2) [2 j * sin (-w0 * t) * sin (-w0 * t)] dt

= (1/T) * integral dari (T/2) ke (-T/2) [2 sin wot * cos (-2 * w0 * t)] dt - (1/T) * integral dari (T/2) ke (-T/2) [2 j * sin (-w0 * t) * sin (-w0 * t)] dt

= 0 - 0

= 0

Jadi, koefisien deret Fourier a-2 untuk sinyal X(t) = 2 sin wot adalah 0.

Untuk menyelesaikan masalah ini, kita perlu melakukan beberapa langkah aljabar.

Pertama, kita dapat menulis ulang (an - bn)³ sebagai (a - b)³n³. Ini menggunakan rumus binomial untuk memperluas pangkat tiga.

Kedua, kita dapat menuliskan a²n sebagai (an)² dan b⁶ sebagai (b²)³.

Ketiga, kita dapat membagi baik pembilang dan penyebut dengan n³ sehingga kita dapat memindahkan pangkat n di dalam kurung.

Dengan melakukan langkah-langkah ini, kita dapat menuliskan ekspresi sebagai berikut:

(a - b)³ (an)² y

a²n² b³ + nx

Sekarang, karena kita ingin mengetahui nilai dari y, kita perlu memanipulasi persamaan sehingga y berada di sisi kiri.

Kita bisa mulai dengan mengalikan kedua sisi dengan a²n² b³ + nx untuk menghilangkan denominasi di bawah garis.

(a - b)³ (an)² y (a²n² b³ + nx)

--------------------------------- = y

a²n² b³ + nx

Ini memberikan kita ekspresi akhir:

y = (a - b)³ (an)²

a²n² b³ + nx

Sekarang kita memiliki persamaan untuk y dalam hal a, b, dan n.
