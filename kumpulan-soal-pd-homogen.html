1. contoh soal dan jawaban PD homogen biar lebih paham


Tentukan penyelesaian dari PD berikut

(x2 – xy + y2) dx – xy dy = 0

Penyelesaian :

Cek terlebih dahulu apakah PD diatas adalah PD homogen

ambil M(x, y) = x2 – xy + y2

M(kx, ky) = (kx)2 – kx ky + (ky)2

= k2(x2 – xy + y2)

N(x, y) = xy

N(kx, ky) = kx ky

= k2(xy)

(x2 – xy + y2) dx – xy dy = 0 adalah PD homogen

(x2 – xy + y2) dx – xy dy = 0, bagi dengan x2, diperoleh

(1 – \frac{y}{x}  + (\frac{y}{x})^2 ) dx – \frac{y}{x}  dy = 0 … (i)

misal : y = ux

dy = u dx + x du

substitusi ke pers (i)

(1 – u + u2) dx – u (u dx + x du) = 0

dx – u dx + u2 dx – u2 dx – ux du = 0

(1 – u) dx – ux du = 0 [bagi dengan x(1 – u)]

\frac{1}{x}  dx – \frac{u}{1-u}  du = 0

\int  \frac{1}{x}  dx – \int  \frac{u}{1-u}  du = c1

ln x – \int  \frac{u-1+1}{1-u}  du = c1

ln x – \int  \frac{u-1}{1-u}  du – \int  \frac{1}{1-u}  du = c1

ln x + u + ln (1 – u) = ln C, dengan ln C = c1

substitusi kembali u = \frac{y}{x} , sehingga

ln x + \frac{y}{x}  + ln (1 – \frac{y}{x} ) = ln C

[tex]\displaystyle (3y-4x)\,dx+(y-x)\,dy=0\\\frac{dy}{dx}=\frac{4x-3y}{y-x}\\\frac{dy}{dx}=\frac{\displaystyle x\left(4-3\cdot\frac{y}x\right)}{\displaystyle x\left(\frac yx-1\right)}\\\frac{dy}{dx}=\frac{\displaystyle 4-3\cdot\frac{y}x}{\displaystyle \frac yx-1}\\\frac{dy}{dx}=f\left(\frac yx\right)\\\\\text{Karena }\frac{dy}{dx}\text{ dapat ditulis sebagai }f\left(\frac yx\right)\text{ maka PD tersebut homogen.}[/tex]

Beberapa konsep yang dipakai:

[tex]\displaystyle \triangleright~\text{PD}:f(x,y)\,dx+g(x,y)\,dy=0\equiv\frac{dy}{dx}=h\left(\frac yx\right)\Rightarrow\text{ PD homogen}[/tex]

18. selesaikanlah soal persamaan diferensial homogen berikut, terima kasihh


Jawaban:

[tex]y = x\sqrt{C_1 x+1}\text{ \: atau} \\ y = - x\sqrt{C_1 x+1}[/tex]

Penjelasan dengan langkah-langkah:

Misalkan y = x.v, maka:

[tex]\frac{dy}{dx} = x\frac{dv}{dx} + v \: \: ...........(i)[/tex]

substitusi y ke pers. pada soal:

[tex]\frac{dy}{dx} = \frac{3(xv)^2 -x^2}{2x(xv)} \\ = \frac{3v^2 - 1}{2v} \ \ ............. (ii)[/tex]

dari (i) dan (ii) diperoleh:

[tex]x\frac{dv}{dx} + v = \frac{3v^2 - 1}{2v} \: \: ............(iii)[/tex]

Selesaikan dv/dx pada (iii):

[tex]\frac{dv}{dx} = \frac{\frac{3v^2-1}{2v} - v}{x} \\

= \frac{\frac{3v^2-1}{2v}-\frac{2v^2}{2v}}{x} \\

= \frac{v^2-1}{2vx} \: \: .......... (iv)[/tex]

Bagi kedua ruas pada (iv) dengan (v^2 - 1)/(2v):

[tex]\frac{\frac{dv}{dx}}{\frac{v^2-1}{2v}} = \frac{1}{x} \\

\frac{2v \: \frac{dv}{dx}}{v^2-1} = \frac{1}{x} \: \: ............(v)[/tex]

integralkan (v) terhadap x:

[tex]\int \left(

\frac{2v \: \frac{dv}{dx}}{v^2-1} \right) \: dx = \int \frac{1}{x} \: dx\\

\ln{(v^2 - 1)}= \ln{x} + C_1 \: \: \: \: \:.......(vi)[/tex]

Karena e ^ ln m = m, maka dengan mengambil e^ dari (vi) diperoleh:

[tex]e^{\ln (v^2-1)} = e^{\ln x + C_1} \\

e^{\ln (v^2 -1)} = e^{\ln x} \: e^{C_1} \\

(v^2-1) = e^{C_1} (x) \\

v^2 = e^{C_1}x+1 \\

v= \pm \sqrt{e^{C_1}x +1}[/tex]

Karena y = x.v, maka:

[tex]y = \pm x\sqrt{e^{C_1} x+1}[/tex]

Catatan:

Cara yang lebih ringkas bisa Anda peroleh dengan menggunakan persamaan Bernoulli.

26. 3. Selesaikan PD tak homogen berikut: Bobot (596)d²yda + 4y = 8 sin 2x​


Materi: Persamaan Diferensial Orde 2 Non Homogen

Penjelasan dengan langkah-langkah:

Mungkin maksudnya begini :

[tex]\frac{d^2y}{dx^2}+4y=8\sin{2x}[/tex]

Karena ini PD tidak homogen, maka solusi umumnya adalah :

[tex]y=y_c+y_p[/tex]

Untuk menemukan yc, gunakan aturan PD homogen.

Akar karakteristik dari persamaan tersebut adalah :

[tex]m^2+4=0\\m^2=-4\\m=±2i[/tex]

Karena akar imajiner, maka kita peroleh a = 0 dan b = 2 sehingga solusi yc kita tuliskan :

[tex]y_c=e^{0x}(C_1\cos{2x}+C_2\sin{2x})\\y_c=C_1\cos{2x}+C_2\sin{2x}[/tex]

Sementara itu, untuk menentukan yp kita gunakan aturan PD tidak homogen, disini kita menggunakan metode variasi parameter saja.

Misalkan :

[tex]y_p=u_1\cos{2x}+u_2\sin{2x}[/tex]

Maka, matriksnya :

[tex]\left(\begin{matrix}\cos{2x}&\sin{2x}\\-2\sin{2x}&2\cos{2x}\end{matrix}\right)\left(\begin{matrix}u_1'\\u_2'\end{matrix}\right)=\left(\begin{matrix}0\\8\sin{2x}\end{matrix}\right)[/tex]

Berdasarkan matriks tersebut, kita bisa menentukan Wronskiannya.

[tex]W=\begin{vmatrix}\cos{2x}&\sin{2x}\\-2\sin{2x}&2\cos{2x}\end{vmatrix}=2\cos^2{2x}+2\sin^2{2x}\rightarrow\,W=2\\W_1=\begin{vmatrix}0&\sin{2x}\\8\sin{2x}&2\cos{2x}\end{vmatrix}=-8\sin^2{2x}\\W_2=\begin{vmatrix}\cos{2x}&0\\-2\sin{2x}&8\sin{2x}\end{vmatrix}=8\sin{2x}\cos{2x}\\W_2=4\sin{4x}[/tex]

Dengan demikian, kita peroleh :

[tex]u_1'=\frac{W_1}{W}=\frac{-8\sin^2{2x}}{2}\\u_1'=-4\sin^2{2x}\\u_1=\int{(-4\sin^2{2x})\,dx}=-4\int{\frac{1}{2}\,dx}+4\int{\frac{1}{2}\cos{4x}\,dx}\\u_1=-2x+\frac{1}{2}\sin{4x}\\u_2'=\frac{W_2}{W}=\frac{4\sin{4x}}{2}\\u_2'=2\sin{4x}\\u_2=\int{2\sin{4x}\,dx}=-\frac{1}{2}\cos{4x}[/tex]

Substitusikan nilai u ini ke persamaan yp tadi, diperolehlah :

[tex]y_p=\left(-2x+\frac{1}{2}\sin{4x}\right)\cos{2x}-\frac{1}{2}\cos{4x}\sin{2x}\\y_p=-2x\cos{2x}+\frac{1}{2}\sin{4x}\cos{2x}-\frac{1}{2}\cos{4x}\sin{2x}[/tex]

Tahap akhir dari solusi tersebut tinggal kita substitusikan dua solusi ini ke solusi umum PD tersebut.

[tex]y=C_1\cos{2x}+C_2\sin{2x}-2x\cos{2x}+\frac{1}{2}(\sin{4x}\cos{2x}-\cos{4x}\sin{2x})\\y=C_1\cos{2x}+C_2\sin{2x}-2x\cos{2x}+\frac{1}{2}\sin{(4x-2x)}\\y=C_1\cos{2x}+C_2\sin{2x}-2x\cos{2x}+\frac{1}{2}\sin{2x}[/tex]

30. tentukan penyelesaian dari pd homogen (x^2+y^2)dx+3x^2ydy=0


Materi : Persamaan Diferensial

Saya bantuin nomor 2 saja y

PD : (x² + y²) dx + 2xy dy = 0

Sebelum menyelesaikannya, periksa dulu apakah pd sudah eksak dengan cara menurunkan parsial.

Jika M = (x² + y²) dan N = 2xy, maka :

&M / &y = 2y

&N / &x = 2y

Karena &M / &y = &N / &x, maka pd eksak.

Selanjutnya,

Jika &u / &y = N, maka u = £ N dy + l(x) sehingga :

u = £ (2xy) dy + l(x)

u = xy² + l(x)...(1)

Turunkan (1) secara parsial terhadap x, sehingga :

&u / &x = M

y² + dl / dx = (x² + y²)

dl / dx = x²

£ dl / dx = £ x²

l(x) = 1/3x³ + c...(2)

Substitusikan persamaan (2) ke (1), didapatlah :

u = xy² + 1/3x³ + c

u - c = xy² + 1/3x³

Karena u - c = k (konstanta), maka :

k = xy² + 1/3x³ (kalikan 3)

3k = 3xy² + x³

x³ + 3xy² - C = 0 dengan C = 3k.

Pembahasan

Materi di alam dapat dibagi menjadi zat tunggal dan campuran. Jika dipelajari lebih lanjut,zat tunggal yang ada di alam juga dapat dikelompokkan menjadi dua jenis yaitu unsur dan senyawa. Berikut pembahasan lebih lanjut terkait zat tunggal dan campuran.

Zat tunggal

Zat tunggal terdiri dari dua jenis yaitu unsur dan senyawa berikut pembahasan terkait unsur dan senyawa:

1) Unsur merupakan zat tunggal yang tidak dapat dibagi lagi menjadi bagian yang lebih sederhana. Banyak sekali unsur yang dapat kalian jumpai di alam diantaranya adalah emas, air raksa, besi, timah, seng, tembaga, dan nikel.  

2) Senyawa merupakan zat tunggal yang dapat diuraikan menjadi dua jenis atau lebih zat yang lebih sederhana dengan cara kimia. Misalnya, air yang memiliki rumus H₂O dapat diuraikan menjadi unsur hidrogen (H₂) dan oksigen (O₂).

Campuran

Campuran merupakan suatu materi yang terdiri dari dua zat atau lebih dan masih mempunyai sifat dari zat asalnya. Campuran dapat dibedakan menjadi dua yaitu campuran homogen dan campuran heterogen.

1) Campuran homogen adalah campuran yang tidak dapat dibedakan zat-zat yang tercampur di dalamnya. Contoh campuran homogen adalah larutan gula dan udara.  

2) Campuran heterogen terjadi karena zat yang tidak dapat bercampur satu dengan lain secara sempurna sehingga dapat dikenali zat penyusunnya. Campuran pasir dan air di dalam gelas merupakan salah satu contoh dari campuran heterogen.

40. selsaikan PD homogen[tex]( x ^ { 3 } + y ^ { 3 } ) d x - 3 x y ^ { 2 } d y = 0 ?[/tex]​


[tex]\large\text{$\begin{aligned}y=\sqrt[\bf3]{\bf\frac{x^3-Cx}{2}}\end{aligned}$}[/tex]

 

Pembahasan

Persamaan [tex]\left(x^3+y^3\right)dx-3xy^2dy=0[/tex]  yang ekuivalen dengan [tex]3xy^2dy=\left(x^3+y^3\right)dx[/tex]  merupakan bentuk persamaan Bernoulli. Namun, penyelesaian di bawah ini menggunakan cara "biasa".

 

[tex]\begin{aligned}&\left(x^3+y^3\right)dx-3xy^2dy=0\\&{\Rightarrow\ }3xy^2dy=\left(x^3+y^3\right)dx\\&{\Rightarrow\ }\left(\frac{3xy^2}{x^3}\right)dy=\left(\frac{x^3+y^3}{x^3}\right)dx\\&{\Rightarrow\ }\left(\frac{3y^2}{x^2}\right)dy=\left(1+\frac{y^3}{x^3}\right)dx\\&{\Rightarrow\ }3\left(\frac{y}{x}\right)^2dy=\left(1+\left(\frac{y}{x}\right)^3\right)dx\end{aligned}[/tex]
[tex]\begin{aligned}&\quad\left[\ \begin{aligned}&\textsf{Ambil $u=\frac{y}{x}$, maka $y=ux$.}\\&\Rightarrow dy=u\,dx+x\,du\end{aligned}\right.\\&{\Rightarrow\ }3u^2(u\,dx+x\,du)=\left(1+u^3\right)dx\\&{\Rightarrow\ }3u^3\,dx+3u^2x\,du=\left(1+u^3\right)dx\\&{\Rightarrow\ }3u^2x\,du=\left(1+u^3-3u^3\right)dx\\&{\Rightarrow\ }3u^2x\,du=\left(1-2u^3\right)dx\\&{\Rightarrow\ }3u^2\cancel{x}\,du\cdot\frac{1}{\cancel{x}}=\left(1-2u^3\right)dx\cdot\frac{1}{x}\end{aligned}[/tex]
[tex]\begin{aligned}&{\Rightarrow\ }3u^2\,du=\frac{1-2u^3}{x}\,dx\\&{\Rightarrow\ }\left(\frac{3u^2}{1-2u^3}\right)du=\frac{1}{x}\,dx\\\\&\quad[\ \ \textsf{Integralkan kedua sisi}\\&{\Rightarrow\ }\int\left(\frac{3u^2}{1-2u^3}\right)du=\int\frac{1}{x}\,dx\\&{\Rightarrow\ }\int\left(\frac{3u^2}{1-2u^3}\right)du=\ln|x|+C_1\end{aligned}[/tex]

[tex].\quad\left[\ \begin{aligned}&\int\left(\frac{3u^2}{1-2u^3}\right)du=3\int\left(\frac{u^2}{1-2u^3}\right)du\\&\textsf{Ambil $v=1-2u^3$}\Rightarrow \frac{dv}{du}=-6u^2\\&\Rightarrow du=\left(-\frac{1}{6u^2}\right)dv\\&\Rightarrow3\int\left(\frac{u^2}{1-2u^3}\right)du=3\int\frac{\cancel{u^2}}{v}\left(-\frac{1}{6\cancel{u^2}}\right)dv\\&\quad=-\frac{3}{6}\int\frac{1}{v}\,dv=-\frac{1}{2}\ln|v|\\\\&\therefore\int\left(\frac{3u^2}{1-2u^3}\right)du=-\frac{1}{2}\ln\left|1-2u^3\right|\end{aligned}\right.[/tex]

[tex]\begin{aligned}&{\Rightarrow\ }-\frac{1}{2}\ln\left|1-2u^3\right|=\ln|x|+C_1\\&{\Rightarrow\ }\ln\left|1-2u^3\right|=-2\ln|x|+C_2\\&\quad[\ \ \textsf{Substitusi $u$ kembali}\\&{\Rightarrow\ }\ln\left|1-\frac{2y^3}{x^3}\right|=-\ln\left|x^2\right|+C_2\\&{\Rightarrow\ }\ln\left|1-\frac{2y^3}{x^3}\right|+\ln\left|x^2\right|=C_2\\&{\Rightarrow\ }\ln\left|x^2\left(1-\frac{2y^3}{x^3}\right)\right|=C_2\\&{\Rightarrow\ }\ln\left|x^2-\frac{2y^3}{x}\right|=C_2\end{aligned}[/tex]
[tex]\begin{aligned}&{\Rightarrow\ }\ln\left|\frac{x^3-2y^3}{x}\right|=C_2\\&{\Rightarrow\ }\ln\left|\frac{x^3-2y^3}{x}\right|=\ln|C|\\&\quad[\ \ \textsf{Keluarkan dari logaritma}\\&{\Rightarrow\ }\frac{x^3-2y^3}{x}=C\\&{\Rightarrow\ }x^3-2y^3=Cx\ \Rightarrow\ 2y^3=x^3-Cx\\&{\Rightarrow\ }y^3=\frac{x^3-Cx}{2}\\&{\therefore\ }\boxed{\ y=\sqrt[3]{\frac{x^3-Cx}{2}}\ }\end{aligned}[/tex]
