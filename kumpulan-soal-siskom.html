3. soal siskom bilangan biner ada yang tauu caranyaa sama jawaban​


4. a.

2F3(16) -> ...(10)

=> 2 x 16² + F x 16¹ + 3 x 16⁰       ( F = 15 )

=> 512 + 240 + 3 = 755(10)

4. b.

3B5(16) -> ...(10)

=> 3 x 16² + B x 16¹ + 5 x 16⁰       ( B = 11 )

=> 768 + 176 + 5 = 949(10)

4. c.

9A2(16) -> ...(10)

=> 9 x 16² + A x 16¹ + 2 x 16⁰       ( A = 10 )

=> 2304 + 160 + 2 = 2466(10)

4. d.

8C3(16) -> ...(10)

=> 8 x 16² + C x 16¹ + 3 x 16⁰      ( C = 12 )

=> 2048 + 192 + 3 = 2243(10)

5. a.

2345(8) -> ...(2)

=> 2 = 010  |  3 =  011  |  4 = 100  |  5 = 101    (masing² diubah ke bentuk biner)

=> 010011100101(2)

5. b.

5432(8) -> ...(2)

=> 5 = 101  |  4 = 100  |  3 = 011  |  2 = 010    (masing² diubah ke bentuk biner)

=> 101100011010(2)

Konversi bilangan desimal (berbasis 10) menjadi bilangan biner (berbasis 2) dapat dilakukan dengan berbagai cara. Salah satunya dengan membagi bilangan desimal tadi dengan angka 2 hingga dihasilkan angka 1.

3086 dibagi 2 hasilnya 1543 sisa 0

1543 dibagi 2 hasilnya 771 sisa 1

48 dibagi 2 hasilnya 24 sisa 0

24 dibagi 2 hasilnya 12 sisa 0

12 dibagi 2 hasilnya 6 sisa 0

6 dibagi 2 hasilnya 3 sisa 0

3 dibagi 2 hasilnya 1 sisa 1

Penulisan konversinya ditulis dari hasil akhir menuju ke atas (angka sisa), sehingga didapatkan bahwa 12345₍₁₀₎ = 11000000111001₍₂₎
