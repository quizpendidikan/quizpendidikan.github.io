4. Tugas sistem operasi saat menerima input dari keyboard dan mouse secara bersamaan adalah mengelola dan memproses input tersebut. Sistem operasi akan mengenali sinyal input dari keyboard dan mouse, memetakan input tersebut ke aplikasi yang sedang aktif, dan mengirim informasi yang relevan ke aplikasi tersebut untuk diolah.

5. Tindakan yang tidak termasuk E-waste (limbah elektronik) adalah:

Memperbaiki dan memperpanjang umur pakai perangkat elektronik yang rusak.

Mendaur ulang perangkat elektronik yang sudah tidak terpakai dengan cara yang aman dan bertanggung jawab.

Mendonasikan perangkat elektronik yang masih berfungsi kepada organisasi atau individu yang membutuhkan.

Menggunakan perangkat elektronik secara efisien dan bertanggung jawab untuk memperpanjang masa pakai dan mengurangi limbah.

6. Harddisk, CD, DVD, USB, dan flash disk merupakan perangkat jeras komputer yang berfungsi sebagai media penyimpanan data.

8. Perangkat yang menyimpan data menggunakan magnetik adalah harddisk. CD, DVD, USB, dan flash disk menggunakan media penyimpanan optik dan flash memory.

9. Perangkat yang dapat membaca data tetapi tidak dapat menulis ulang adalah CD-ROM (Compact Disc-Read Only Memory).

10. Komputer terdiri dari perangkat keras yang meliputi CPU (Central Processing Unit), motherboard, RAM (Random Access Memory), harddisk, monitor, dan keyboard. Namun, tidak termasuk dalam pertanyaan.

11. Perangkat lunak yang terhubung ke komputer termasuk perangkat lunak aplikasi seperti program pengolah kata, program pengedit foto, program penjelajah web, dll.

12. Elemen-elemen komputer yang tidak termasuk dalam pertanyaan adalah:

Listrik (komputer membutuhkan pasokan listrik untuk beroperasi, tetapi bukan elemen komputer itu sendiri).

Lingkungan kerja (seperti meja, kursi, dan ruangan tempat komputer berada).

13. Perangkat yang memiliki slot untuk menempelkan perangkat lainnya adalah motherboard komputer. Slot tersebut digunakan untuk memasang kartu ekspansi seperti kartu grafis, kartu suara, dll.

14. Jenis memori yang hanya bisa membaca data saja adalah ROM (Read-Only Memory). Data yang tersimpan di dalam ROM tidak dapat diubah atau ditulisi ulang.

15. Antarmuka yang menggunakan karakter untuk memberikan perintah adalah antarmuka baris perintah (command-line interface). Contohnya adalah Command Prompt pada Windows atau Terminal pada Linux.
