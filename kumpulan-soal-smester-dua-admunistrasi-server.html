12. Kumpulan server client yang menggunakan server ketika menjalankan validasi data untuk login pada jaringan disebut


Kumpulan server client yang menggunakan server ketika menjalankan validasi data untuk login pada jaringan disebut domain

Pembahasan

Jaringan komputer adalah jarinagn yang digunakan untuk menghubungkan komunikasi antar komputer. Komputer client dan server adalah perangkat keras yan dibutuhkan untuk membangun sebuah jaringan pada komputer.

Topologi jaringan pada komputer, sebagai berikut:

- Topologi bus

- Topologi bintang (star)

- Topologi cincin (ring)

- Topologi mesh

- Topologi pohon (tree)

- Topologi linier


Fungsi domain

- Memudahkan usermencari alamat yang dituju.

- Menjadi identitas dari websute tertentu.

- Domain yang unik akan mudah diiingat oleh user.


Contoh domain

- .net

Domain ini digunakan pada organinasi pada bidang teknologi jaringan. Misal telkom.net

- .com

Domain ini digunakan untuk organisasi yang komersial. Misal google.com, youtube.com

- .org

Domain ini digunakan untuk organisasi non profit yaitu sekolah, komunitas dan sebagainya. Misalnya wikipedia.org, opensource.org


Perbedaan komputer client dan server

- Komputer client

Komputer yang digunakan untuk meminta layanan kepada server berupa gambar, data, file dan akses lainnya.

- Komputer server

Komputer yang digunakan untuk menampung semua data yang berasal dari komputer client pada sebuah jaringan.


Fungsi komputer client dan server

- Komputer client

a. Menjalankan sebuah program secara maksimal karena tidak terganggu dengan tugas lainnya.

b. Digunakan untuk mengakses beberapa data yang terdapat dalam komputer server.

c. Menjadi sistem keamanan administrasi bagi perusahaan atau instansi pemerintah lainnya.

- Komputer server

a. Melayani permintaan semua komputer client.

b. Menyediakan sumber resource berupa software, hardware, dan aplikasinya.

c. Menyimpan file atau data yang dapat diakses secara bersamaan oleh komputer server maupun semua komputer client.

d. Mengatur hak akses dalam sebuah jaringan.

15. Apa yang di maksud dengan client dan server?jelaskan perbedaan keduanya!


Komputer server adalah komputer yang digunakan untuk menampung semua data yang berasal dari komputer client dalam sebuah jaringan. Tidak hanya itu, komputer server juga digunakan untuk mengelola sebuah jaringan komputer. Sirkulasi data, informasi, maupun perintah yang berasal dari komputer client akan selalu melewati komputer server ini.

Komputer client adalah komputer yang digunakan untuk meminta layanan tertentu dari komputer server. Layanan tersebut bisa jadi data, file, gambar, printer, maupun yang lainnya. Oleh karena itu, di dalam komputer client dibutuhkan suatu aplikasi tertentu agar dapat mengakses layanan dari komputer server.

Perbedaannya :

Komputer server berfungsi untuk menyimpan seluruh data, melayani permintaan akses dari komputer client, serta mengelola lalu lintas data dalam sebuah jaringan. Sedangkan, komputer client meminta layanan dari komputer server untuk bisa mengakses sebuah data.

Komputer server dapat melakukan beberapa layanan yang dibutuhkan komputer client secara bersamaan. Sedangkan, komputer client tidak mempunyai layanan yang dapat diakses oleh komputer client lain.

Client dan Server merupakan sebuah teknologi jaringan dimana Server bertugas melayani permintaan data dari Client, Server merupakan pusat pengolahan data sedangkan Client menyediakan interface yang berhubungan langsung dengan User yang selanjutnya meneruskan permintaan data kepada Server

29. Banatu bang soal mtk kelas 8 smester satu ujian harian


Garis yang sejajar memiliki gradien yang sama

Garis k melalui (0,5) dan (-3,0) Garis l melalui (3,0) dan (0,-5)

.

Gradien garis k = Gradien Garis l

[tex] \tt \frac{y2 - y1}{x2 - x1} = \frac{y2 - y1}{x2 - x1} [/tex]

[tex] \tt \frac{0 - 5}{ - 3 - 0} = \frac{ - 5 - 0}{0 - 3} [/tex]

[tex] \tt \frac{ - 5}{ - 3} = \frac{ - 5}{ - 3} [/tex]

[tex] \tt \frac{5}{3} = \frac{5}{3} [/tex]

Maka, garis k dan garis l terbukti sejajar.
