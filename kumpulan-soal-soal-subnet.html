1. Soal IP Address 192.168.1.0/29 Tentukan: 1.subnet mask 2.jumlah subnet/network 3.jumlah host per subnet 4.blok subnet​


1. Subnet Mask: /29 berarti hanya 29 bit yang dirubah dari alamat IP menjadi subnet address. Jadi subnet mask untuk IP address 192.168.1.0/29 adalah 255.255.255.248.

2. Jumlah Subnet/Network: Dalam /29 ada 3 bit host, jadi jumlah subnet adalah 2^3 = 8.

3. Jumlah Host per Subnet: Dalam /29 hanya ada 3 bit host, jadi jumlah host per subnet adalah 2^3 - 2 = 6. Karena 2 host digunakan untuk network address (192.168.1.0) dan broadcast address (192.168.1.7).

4. Blok Subnet: Diketahui IP address 192.168.1.0/29 dan subnet mask 255.255.255.248. Blok subnet akan tergantung pada range IP address yang bisa digunakan dalam subnet tersebut.

IP address pertama dalam subnet: 192.168.1.0

IP address terakhir dalam subnet: 192.168.1.7

Sehingga blok subnet adalah 192.168.1.0 - 192.168.1.7

4. Pada IP 196.189.200.1/26 tentukanjumlah subnetjumlah host persubnetblok subnet​


Jawaban:

IP 196.189.200.1/26 merupakan alamat IP dengan subnet mask 255.255.255.192 atau dalam bentuk CIDR /26. Subnet mask tersebut menunjukkan bahwa terdapat 26 bit yang direservasi untuk network ID dan sisa bit (6 bit) digunakan untuk host ID.

Jumlah Subnet

Jumlah subnet dapat dihitung dengan menggunakan formula 2^n dimana n adalah jumlah bit yang direservasi untuk subnet. Pada kasus ini, terdapat 6 bit yang direservasi untuk subnet, sehingga jumlah subnet yang dapat dibentuk adalah 2^6 = 64 subnet.

Jumlah Host Persubnet

Jumlah host persubnet dapat dihitung dengan menggunakan formula 2^n-2 dimana n adalah jumlah bit yang direservasi untuk host. Karena pada kasus ini terdapat 6 bit yang direservasi untuk host ID, maka jumlah host persubnet yang dapat dihitung adalah 2^6-2 = 62 host.

Blok Subnet

Untuk menentukan blok subnet, kita perlu menghitung nilai dari setiap oktet pada alamat IP dan subnet mask. Berikut adalah nilai dari setiap oktet pada alamat IP dan subnet mask:

Alamat IP: 196.189.200.1 = 11000100.10111101.11001000.00000001

Subnet mask: 255.255.255.192 = 11111111.11111111.11111111.11000000

Untuk menentukan blok subnet, kita cukup melakukan operasi bitwise AND antara alamat IP dan subnet mask. Berikut adalah hasil operasi bitwise AND tersebut:

11000100.10111101.11001000.00000001 (IP Address)

11111111.11111111.11111111.11000000 (Subnet Mask)

11000100.10111101.11001000.00000000 (Network ID)

Berdasarkan hasil tersebut, blok subnet untuk alamat IP 196.189.200.1/26 adalah 196.189.200.0/26.


5. jumlah subnet dan host per subnet dari ip 192.168.1.1/19 berapa?


Jumlah subnet dan host per subnet dari IP 192.168.1.1/19 adalah sebagai berikut:

- Notasi CIDR /19 menandakan bahwa 19 bit pertama pada alamat IP tersebut digunakan untuk network ID, sehingga sisa 13 bit digunakan untuk host ID.
- Jumlah subnet dapat dihitung dengan rumus 2^k, di mana k adalah jumlah bit yang digunakan untuk network ID. Dalam hal ini, k = 19, sehingga jumlah subnet adalah 2^19 = 524,288 subnet.
- Jumlah host per subnet dapat dihitung dengan rumus 2^h - 2, di mana h adalah jumlah bit yang digunakan untuk host ID. Dalam hal ini, h = 13, sehingga jumlah host per subnet adalah 2^13 - 2 = 8,190 host per subnet.

Jadi, IP 192.168.1.1/19 memiliki 524,288 subnet dan 8,190 host per subnet.

11. Jumlah subnet,jumlah host persubnet dan blok subnet dari 191.198.1.1/26


Jawaban:Jika menggunakan subnet mask /26 pada alamat IP 191.198.1.1, maka jumlah subnet yang dihasilkan adalah 2 (dua). Jumlah host per subnet adalah 62, sedangkan blok subnet adalah 64.

Penjelasan:

Untuk menghitung jumlah subnet, host per subnet, dan blok subnet, pertama-tama Anda perlu mengetahui jumlah bit yang digunakan untuk menyatakan subnet mask. Dalam kasus /26, terdapat 26 bit yang digunakan untuk menyatakan subnet mask, sehingga sisa bit yang tersisa untuk menyatakan host adalah 32 - 26 = 6 bit.

Dengan menggunakan 6 bit untuk menyatakan host, maka jumlah host per subnet adalah 2^6 = 64. Karena setiap subnet harus menyisakan satu alamat untuk menyatakan network address dan satu alamat lainnya untuk menyatakan broadcast address, maka jumlah host yang sebenarnya tersedia per subnet adalah 64 - 2 = 62.

Blok subnet adalah jumlah alamat yang terdiri dari network address, host address, dan broadcast address. Dengan menggunakan 6 bit untuk menyatakan host, maka blok subnet adalah 2^6 = 64.

Dengan demikian, jumlah subnet yang dihasilkan dari alamat IP 191.198.1.1 dengan subnet mask /26 adalah 2, jumlah host per subnet adalah 62, dan blok subnet adalah 64.

Jawaban:

a. Subnet Mask:

IP Network 192.168.12.0/26 memiliki subnet mask 255.255.255.192. Ini dapat dihitung dengan mengonversi "/26" menjadi notasi biner (11111111.11111111.11111111.11000000) dan kemudian mengonversi notasi biner tersebut menjadi notasi desimal, yang menghasilkan 255.255.255.192.

b. Jumlah Subnet:

Dalam /26, kita memiliki 6 bit untuk host, sehingga jumlah subnet yang mungkin adalah 2^6 = 64 subnet.

c. Jumlah Host per Subnet:

Dalam /26, kita memiliki 6 bit untuk host. Namun, 2 alamat (alamat jaringan dan broadcast) tidak dapat digunakan untuk host. Jadi, kita punya 2^6 - 2 = 62 host per subnet.

d. Blok Subnet:

Setiap subnet akan memiliki rentang IP dari alamat jaringan hingga alamat broadcast. Dalam kasus ini, kita memiliki 62 host per subnet, sehingga blok subnet pertama akan dari 192.168.12.0 hingga 192.168.12.63, blok subnet kedua dari 192.168.12.64 hingga 192.168.12.127, dan seterusnya.

e. Tabel Alamat Host dan Broadcast Valid:

Untuk setiap subnet, alamat pertama (alamat jaringan) dan alamat terakhir (alamat broadcast) tidak dapat digunakan untuk host. Jadi, untuk subnet pertama, alamat host yang valid adalah dari 192.168.12.1 hingga 192.168.12.62, dan alamat broadcast adalah 192.168.12.63. Begitu seterusnya untuk subnet berikutnya.

b. Host Per Subnet:

Jumlah host per subnet dapat dihitung menggunakan rumus 2^n−2, di mana n adalah panjang maska subnet dalam bit. Dalam kasus ini, n=18, sehingga host per subnet adalah 2^18−2=262,142 host.

c. Alamat Subnet Mark:

Alamat subnet mark adalah alamat IP pertama dalam subnet. Dalam notasi IP, bit host yang terakhir akan diatur ke 0. Jadi, alamat subnet mark adalah 172.16.64.0.

d. Blok Subnet:

Blok subnet adalah kisaran alamat IP yang dapat dialokasikan dalam subnet. Dalam kasus ini, dengan panjang maska 18 bit, blok subnet dimulai dari alamat subnet mark (172.16.64.0) hingga alamat broadcast terakhir dalam subnet.
