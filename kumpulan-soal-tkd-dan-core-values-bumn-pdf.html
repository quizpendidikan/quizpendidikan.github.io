1. Dukungan apa yang dibutuhkan oleh SDM BUMN agar proses internalisasi core values AKHLAK dan 18 panduan perilaku dapat berjalan secara optimal?


Untuk memastikan nilai-nilai inti dan pedoman perilaku AKHLAK terinternalisasi dengan baik, SDM BUMN membutuhkan bantuan dari atasan, pelatihan dan sosialisasi, budaya dan sistem perusahaan, serta pemantauan dan evaluasi.

Pembahasan

Nilai-nilai utama atau core values BUMN adalah AKHLAK yang merupakan singkatan dari Amanah, Kompeten, Harmonis, Loyal, Adaptif, dan Kolaboratif. Amanah yang Memegang teguh kepercayaan yang diberikan.

Nilai Inti itu penting, dan setiap orang di tempat kerja  harus menjalankannya. Nilai-nilai tersebut harus diikuti oleh Direksi, Dewan Komisaris/Dewan Pengawas, manajemen/karyawan, dan karyawan/pekerja di lingkungan Badan Usaha Milik Negara, Anak Perusahaan dan Perusahaan Afiliasi Konsolidasi.

17. yang tidak tetmasuk panduan perilaku loyal dalam core values ASN


Yang tidak termasuk panduan perilaku loyal dalam core values ASN ialah hal-hal yang tidak berkaitan atau berlawanan dengan perilaku berdedikasi tinggi terhadap kepentingan bangsa dan negara. Core Values ASN adalah nilai mutlak yang harus dilaksanakan oleh para ASN di seluruh Indonesia.

Pembahasan

Sebagai Lembaga pemerintahan yang menjalankan fungsi Manajemen ASN, Badan Kepegawaian Negara memiliki peranan yang sangat penting dalam mendukung terciptanya ASN yang BerAKHLAK (Berorientasi Pelayanan, Akuntabel, Kompeten, Harmonis, Loyal, Adaptif, dan Kolaboratif). Panduan Perilaku Core Values ASN ini disusun sebagai upaya membentuk budaya BerAKHLAK sebagai identitas dan ciri khas bagi ASN di lingkungan Badan Kepegawaian Negara, serta memberikan arah bagi para ASN dalam menjalankan tugas dan fungsinya sesuai dengan Perundang-Undangan yang berlaku.

Panduan Perilaku BerAKHLAK ini bertujuan untuk memberikan arah kepada seluruh ASN dalam berperilaku, memberikan pelayanan, maupun melakukan pekerjaannya seharisehari. Dengan adanya Panduan Perilaku ini, diharapkan para Aparatur Sipil Negara mampu menanamkan nilai-nilai BerAKHLAK dalam melaksanakan fungsinya sebagai seorang pelaksana kebijakan, pelayan publik, serta perekat dan pemersatu bangsa secara bertanggung jawab dan berintegritas.

Jawaban:

Inti memiliki arti “inti.” Istilah “core” dipakai buat menentukan jumlah core (jumlah core) dari suatu prosesor, baik CPU dan GPU.

Jumlah inti adalah yang menggunakan. Biasanya digunakan untuk mendefinisikan istilah “prosesor dual-core”, “prosesor quad-core”, “hexa core” dan “octa-core”, mis.

Prosesor inti tunggal = 1 inti

Prosesor dual core = 2 core

Quad core = 4 core

Hexa Core Processor = 6 Core

Prosesor Octa Core = 8 core

CPU Deca Core = 10 Core

Dual Core adalah penggunaan dua buah inti prosessor dalam sebuah kemasan prosesor konvensional. (Contoh: Intel Celeron N4020)

Quad Core adalah penggunaan empat buah inti prosessor dalam sebuah kemasan prosesor konvensional. (Contoh: Intel i7-860)

Octa Core adalah penggunaan empat buah inti prosessor dalam sebuah kemasan prosesor konvensional. (Contoh: Intel Core i3 7100 7th)

Penjelasan:

Perbandingan antara 3 core tersebut hanya terdapat pada jumlah inti prosesornya. Semakin banyak core, semakin kencang performa yang dihasilkan.
