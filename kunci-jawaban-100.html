a + b ≤ 100
5.000a + 10.000b ≤ 650.000
a + 2b ≤ 130

2.000a + 3.000b = ...

a + b = 100
Jika a = 0, b = 100 (0,100)
Jika b = 0, a = 100 (100,0)

a + 2b = 130
Jika a = 0, b = 65 (0,65)
Jika b = 0, a = 130 (130,0)

a + b = 100
a + 2b = 130
Eliminasi a
-b = -30
b = 30

a + 30 = 100
a = 70

2.000(0) + 3.000(100) = 300.000 --> Untung max
2.000(130) + 3.000(0) = 260.000
2.000(70) + 3.000(30) = 230.000

Jadi, pengrajin akan mendapat keuntungan max jika memproduksi jenis 2 sebanyak 100 buah dan jenis 1 sebanyak 0 buah
